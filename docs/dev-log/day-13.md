## 🧠 学び (= 副次知見、Day 13+ 観測候補)

### 副次知見 1: 不発閾値の非対称性 = 観測装置側要因の切り分け余裕

> ぜんぶ忘れていい、この 1 シーンだけ。
> 「3 例で確定 vs 5 例で却下」 の差は **観測装置の故障切り分け時間** を見ているだけ。実例 1 件あれば「動く装置」 と分かるが、何も起こらない時は「装置が壊れているのか仮説が間違いか」 区別がつかない、だから 2 例余分に観測する。

確定側 (= 3 例観測) の根拠 = Day 11-4 9 ステップフロー再現性確定 (= 3 例で機械的再現が言える)。

不発側 (= 5 例観測) の根拠 = 観測装置側要因 vs 仮説側要因の切り分けに余裕が要る。3 例不発の時点では装置設計ミス (= タイミング不一致 / セッション中断 / 仕様変更) を疑う段階、追加 2 例不発で「装置は健全、仮説が間違い」 が言える段階。

→ 「非対称性 = サンプル数の倍率」 ではなく **「不発の意味解釈に時間を払う」** という運用境界。

### 副次知見 2: 集計慣習の事前確定範囲 = 不発側まで含めて事前固定

Day 12-2 PR #293 で集計慣習の事前確定を確立した時点では「確定側 (= 3 例観測達成) を事前確定」 までだった。不発側は「不発が継続したら却下方向に記録」 と曖昧な表現で残っていた (= 不発閾値 N が未定義)。

→ 本 PR で **不発側まで事前確定範囲を拡張** = 「結果が出てから却下逃れバイアス」 を不発側でも回避。事前確定は確定側だけでなく不発側にも適用する原則を memory に明記。

### 副次知見 3: Issue 起票表現の memory 化時整理 = 副局長補正の実例

Issue #294 起票時の「ブラックスワン論理逆応用」 は方向感を誤る表現 (= ブラックスワン論理は確証の非対称性、本文脈では「類似」 が正確)。memory 化時に副局長が整理して書き換え = **Issue は仮説 / PR は確定、上書き可** ルールの自己適用。

→ 自己提唱の言い切りを含まない範囲の格下げ (= 学び 36 自己完結型パターン的)、ただし memory 整理 PR の範疇なのでサブカテゴリ 3 のサンプル 2 例目として記録。
