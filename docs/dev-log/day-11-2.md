### 副次知見 2: Issue #N (CLOSED) の決着判断ログを後続 PR description で補強する運用パターン

strategic-reviewer 論点 1 由来。Issue #235 (= 唐揚げ系ご褒美化リスク対応) は close コメントゼロで CLOSED 状態だったが、**本 PR description で「案 B (= 言語的緩和) を過去形語尾の補助コピーで部分実装」 と決着判断ログを補強**。

→ 「リスク認識を Issue 本文に残す → close 時にコメントなし」 だと後輩が決着を辿れない。後続 PR description で **「Issue #N (CLOSED) との関係」 セクションを設けて関係性を 1 行明記** すれば、後輩が「リスク → 対処 → 残った内在リスク」 のチェーンを 1 本で辿れる教材になる。

How to apply:

- close 済 Issue でも **後続 PR で関係する判断を補強する場面** があれば PR description に「Issue #N (CLOSED) との関係」 セクション追加
- close 時のコメント 1 行運用は理想だが、漏れる場合は後続 PR で吸収する代替パスとして有効
- 学び 36 = 外部レビューによる再相対化 の延長で「**Issue 間の因果関係も再相対化対象**」 として運用化候補 (= 副局長 strategic-reviewer 学びポイント由来)
